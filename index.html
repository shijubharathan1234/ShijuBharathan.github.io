Airbnb tableau project

I conducted a data analysis project on Airbnb listings using Tableau to identify trends and insights that could help hosts optimize their pricing and improve listing performance. By creating interactive dashboards, I visualized key metrics such as average bedroom prices, and price distribution across zipcodes. The analysis revealed factors influencing pricing, including location, number of bed, number of bedrooms and their demand, for hosts to maximize revenue based on market trends.
